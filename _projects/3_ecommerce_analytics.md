An end-to-end data analytics platform for processing and visualizing large-scale e-commerce data.

Technical stack: PySpark, React, SQL

Highlights:

- Built a PySpark ETL pipeline to ingest, clean, and transform a 5GB e-commerce dataset into analytics-ready tables
- Applied join optimization and DataFrame caching to improve query performance
- Developed a React dashboard with date, category, and region filtering, trend charts, and drill-down views to surface product and sales analytics
